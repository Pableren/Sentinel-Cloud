# Habilidad: Dashboard & Visualización

## 📝 Descripción
Esta habilidad se encarga de la capa de presentación de datos. Permite visualizar los insights generados por el pipeline de Big Data y el rendimiento de los modelos.

## 🎯 Objetivos
*   Proveer visualizaciones claras y accionables.
*   Conectar directamente con los datos procesados (Parquet/MinIO) o a través de Trino.
*   Monitorizar métricas de negocio y de sistema.

## 🛠️ Tareas Comunes
1.  **Desarrollo de Gráficos:**
    *   Crear series temporales de telemetría.
    *   Visualizar comparativas de predicciones vs valores reales.
2.  **Conexión de Datos:**
    *   Leer datos eficientemente desde el Data Lake.
3.  **UI/UX:**
    *   Asegurar una interfaz responsive y amigable.

## 📦 Estructura de Archivos Relacionada
*   `src/dashboard/`: Código fuente de la aplicación de dashboard (ej. Dash, Streamlit o Custom Web App).
